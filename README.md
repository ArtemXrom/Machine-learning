# Handwritten Digit Recognition Neural Network

Нейронная сеть для распознавания рукописных цифр из датасета MNIST с возможностью тестирования на пользовательских изображениях.

## 📋 Описание проекта

Проект реализует простую нейронную сеть прямого распространения с одним скрытым слоем для классификации рукописных цифр (0-9). Сеть обучается на датасете MNIST и может распознавать цифры на пользовательских изображениях.

## 🏗️ Архитектура сети

- **Входной слой**: 784 нейрона (28×28 пикселей)
- **Скрытый слой**: 20 нейронов
- **Выходной слой**: 10 нейронов (по одному на каждую цифру)
- **Функция активации**: Сигмоида
- **Функция потерь**: MSE (Mean Squared Error)
